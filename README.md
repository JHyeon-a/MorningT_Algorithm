## 3. 진행방식
- 스터디원은 각자 매주 공통 문제 2문제 + 개인 문제 2문제를 풀이한다.
- 알고리즘 수업 이후 매주 한 명씩 알고리즘 개념 발표
- 공통 문제 -> 두 명 이상 못 푼 경우 코드 리뷰 진행
- 개인 문제 -> 풀었을 경우 풀이 방식 설명, 못 푼 경우 본인이 시도한 접근법 공유
***
## 4. 스터디 룰
- 화, 목 07:50 ~ 08:50
  - 08:00부터 지각비
  - 지각비는 1,000원으로 시작, 피보나치로 증가
- 문제 안 푼 경우
  - 벌금 3,000원
  - 몰라서 못 푼 경우, 해당 문제 접근법에 대한 주석 등 코멘트 달기
